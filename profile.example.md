# Profil utilisateur

> Ce fichier est créé automatiquement par `/init`. Vous pouvez le modifier à tout moment.
> Il est lu par les agents et les commandes pour personnaliser les recherches et les publications.
> **Ce fichier est ignoré par git** (données personnelles).

## Identité

- **Nom** : Prénom Nom
- **Rôle** : votre fonction ou titre
- **Organisation** : nom de votre entreprise ou structure

## Contexte professionnel

> C'est la section la plus importante. Elle sert de **filtre de pertinence** pour toutes vos recherches.
> Décrivez en 2-3 phrases ce sur quoi vous travaillez et ce qui guide votre curiosité intellectuelle.

Exemple : "Je travaille sur l'innovation et l'introduction de l'IA générative dans les entreprises (ETI, grandes entreprises, startups). Mes recherches doivent toujours être filtrées par cette lentille : est-ce pertinent pour mes projets ? Est-ce que ça change quelque chose à ma compréhension du monde ?"

## Canaux de publication

> Configurez ici les canaux sur lesquels vous publiez. Chaque canal a son ton, son format et ses contraintes.
> Les canaux sont utilisés par la commande `/publish`.
> Supprimez les sections des canaux que vous n'utilisez pas.

### Newsletter

- **Nom** : nom de votre newsletter
- **Ton** : décrivez le ton (ex: personnel, en "je", pédagogique mais pas professoral, curieux)
- **Longueur cible** : 800-1200 mots
- **Exemples de phrases typiques** :
  - "..."
  - "..."
- **À éviter** :
  - pas de jargon académique non expliqué
  - pas de ton corporate

### Blog

- **Nom** : nom de votre blog ou site
- **Ton** : décrivez le ton (ex: expert accessible, structuré, engageant)
- **Longueur cible** : 1500-2500 mots
- **À éviter** :
  - ...

### LinkedIn

- **Ton** : décrivez le ton (ex: direct, affirmé, un point de vue pas un résumé)
- **Longueur cible** : 150-250 mots
- **Exemples de posts qui ont bien marché** :
  - "..."
- **À éviter** :
  - pas de hashtags à rallonge
  - pas de ton "coach motivationnel"
  - pas d'émojis en début de chaque ligne

### Outil design (carrousels)

- **Outil** : Canva / Figma / autre
- **Préférences visuelles** : couleurs, style, police
- **Slides** : 6-10 slides, 20-40 mots par slide
